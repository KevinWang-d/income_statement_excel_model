Income Statement Financial Model (Excel)

Overview
This project is a multi-sheet Excel financial model built to practice income statement consolidation and financial analysis.

It demonstrates how raw accounting data can be transformed into structured financial reports.

Purpose
- Practice Excel financial modeling
- Learn formula-based consolidation
- Build linked income statements
- Analyze performance across periods

Tools Used
- Microsoft Excel
- Financial Modeling
- Advanced Formulas
- Data Validation

Workbook Structure
- YEAR: Time and period controls
- DataSource: Raw financial data
- REFERENCE / SPECIAL ACCOUNTS: Mapping tables
- TY IS: Current year income statement
- NY IS: Next year income statement

 What I Did
- Designed multi-sheet reporting structure
- Built dynamic formulas and references
- Created consolidation logic
- Implemented account grouping rules
- Developed period comparison model

 Key Learning Outcomes
- Improved financial modeling skills
- Learned consolidation techniques
- Strengthened Excel formula proficiency
- Built structured reporting templates

How to View
- Download and open with Microsoft Excel
